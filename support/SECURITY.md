# セキュリティ報告 / Security Reporting

セキュリティ上の問題は、公開 GitHub Issue に投稿しないでください。<br>
Do not report security issues in public GitHub issues.

OceanGraph のセキュリティ上の問題を見つけた場合は、以下の連絡先に報告してください。<br>
If you believe you have found a security issue in OceanGraph, contact:

`support(at)unvelyze.com`

問題の内容と、再現または理解に必要な情報を記入してください。<br>
Include a clear description of the issue and enough detail to reproduce or understand it.

パスワード、API キー、トークン、セッション Cookie、非公開のユーザーデータは、非公開チャネルで明示的に依頼された場合を除き含めないでください。<br>
Do not include passwords, API keys, tokens, session cookies, or private user data unless explicitly requested through a private channel.

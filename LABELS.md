# Issue Labels

Create these labels in the `Lot4Fun/oceangraph-support` repository.

| Label | Color | Description |
| --- | --- | --- |
| `needs triage` | `#FBCA04` | New issue that has not been reviewed yet. |
| `bug` | `#D73A4A` | Something is not working as expected. |
| `feature request` | `#A2EEEF` | Request for new or improved functionality. |
| `question` | `#D876E3` | Usage question or support request. |
| `documentation` | `#0075CA` | Documentation issue or improvement. |
